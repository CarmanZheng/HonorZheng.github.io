### 第二部分 浮动和布局

#### 1. 浮动 

浮动用到了css里面的float属性

| 属性值  | 描述                     |
| :------ | ------------------------ |
| none    | 元素不浮动，标准流布局   |
| left    | 元素浮动在父级元素的左边 |
| right   | 元素浮动在父级元素的右边 |
| inherit | 继承父级元素的float属性  |

##### 1.1 浮动可以是块级元素在同一行显示

```html
<div style="float:left">
    <div style="background:red">
        块级元素A
    </div>
    <div style="background:yellow">
        块级元素B
    </div>            
</div>
```

##### 1.2 浮动使行内元素具有块级元素特征

tip：标准流排版中，行内元素不能设置**宽、高、上下边距**；

当设置行内元素为浮动时，其具有块级元素的特征，可以设置**宽、高和四个方向的边距**

```html
<style>
    span{
        float:left;
        width:80px;
        height:80px;
        margin:15px;
        padding:15px;
    }
</style>

<div>
    <span>span1</span>
    <span>span2</span>
</div>
```
